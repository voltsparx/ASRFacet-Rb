# `lib/asrfacet_rb/lab/`

Safe local validation targets live here.

## Purpose

This folder provides intentionally simple local targets that help users verify
ASRFacet-Rb behavior before running against real authorized environments.

## What belongs here

- local demo services
- benign vulnerable-pattern simulators
- fixtures that help exercise crawling, header checks, JS extraction, and
  reporting behavior

## Important note

These lab targets are for framework validation and education. They should stay
sanitized and must not contain real secrets, malware, or unsafe destructive
behavior.

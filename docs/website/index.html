ASRFacet-Rb
MULTI-STAGE · OPERATOR-FOCUSED · DOCUMENTED LIKE A REAL TOOL
ASRFacet-Rb is a Ruby 3.2+ attack surface reconnaissance framework for authorized testing. It combines passive discovery, DNS and certificate enrichment, service mapping, web analysis, report generation, and guided operator UX in one place.

⚠ Authorized Use Only
Use ASRFacet-Rb only on systems you own or have explicit written permission to test. The framework includes scope controls, recon memory, and guided explanations to keep operators inside approved boundaries.

Core Capabilities
🔍
Passive Discovery
Uses multiple passive sources before any active probe hits the target.
🌐
DNS & Cert Pivoting
Validates hostnames and expands the asset set through DNS and SAN pivots.
📡
Service Mapping
Finds reachable services with threaded TCP scanning and banner collection.
🕷️
Web Recon
Performs HTTP probing, crawl analysis, JavaScript endpoint extraction, and correlation.

Operator Guidance
